# Project Brief — TennisBootcamp.ca

## Project Name
TennisBootcamp.ca

## Goal
Build a premium tennis training site with intake-driven lead capture. Every design and copy decision should serve conversion, trust, and athletic credibility.

## Primary Conversion
**Get Priority Placement** → `/intake`

## Secondary Conversion
Newsletter signup

## Current Working Systems
- Intake form (`/intake` page)
- API route for form submission (`/api/intake`)
- Google Sheets integration for lead storage
- Lead scoring logic based on intake responses
- Apps Script for formatting and sorting leads in the spreadsheet

## Current Focus
- Hero section visual fidelity
- Homepage trust and conversion signals

## Non-Negotiables
1. Do not break the intake flow under any circumstances
2. No fake design approximations — when an exact reference is required, use it
3. Keep the site athletic, premium, and clean at all times
4. All changes must be tested against the intake form pipeline before being considered complete
